TOPAL WORLD BACKGROUND PATCH v3

This patch updates the animated site background again.

Main changes in v3:
- uses the newer refined building illustrations
- Anderson Hall now includes surrounding plants and terraces
- makes the entire background much more transparent so page text is easier to read
- reduces the size of the building illustrations
- makes the world map lines cleaner and easier to read
- changes the electric travel lines into more continuous nerve-like connection paths
- keeps the flat-map to globe scroll transition, but in a subtler way

Files included:
- _layouts/default.html
- assets/css/world-journey-background.css
- assets/js/world-journey-background.js
- assets/img/background/sorbonne-line.png
- assets/img/background/anderson-hall-line.png
- assets/img/background/old-main-line.png

Install by replacing the files on your site with the files in the same paths from this patch.
