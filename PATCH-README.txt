TOPAL WORLD BACKGROUND PATCH

This patch adds a site-wide animated academic journey background.

Files included:
- _layouts/default.html
- assets/css/world-journey-background.css
- assets/js/world-journey-background.js

What it does:
- Adds a large outlined world map behind the site
- Places three smaller line-art buildings at the correct cities:
  - Old Main in Tucson / Arizona / southwestern United States
  - Sorbonne-associated building in Paris / France
  - Anderson Hall in Istanbul / Turkey
- Adds glowing city dots and animated connection lines
- On scroll, the flat map moves downward and fades into a rotating globe
- Applies globally across pages

Install:
1. Copy each file into the same path in your site.
2. Rebuild or redeploy the site.

Notes:
- No existing image files were re-added.
- The background is code-based and uses inline SVG, CSS, and JS only.
